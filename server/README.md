# FluxERP

Sistema SaaS para gestão de pequenos negócios.

## Funcionalidades

- Autenticação com JWT
- Cadastro de empresas
- Cadastro de produtos
- Cadastro de clientes
- Categorias
- Controle de estoque
- Vendas
- Dashboard

## Tecnologias

- Node.js
- Express
- PostgreSQL
- JWT
- Bcrypt
- Dotenv
- CORS
- Nodemon
